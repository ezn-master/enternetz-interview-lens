How to Use — उपयोग कैसे करें
1
Fill candidate details
उम्मीदवार का नाम और पद भरें — यह ज़रूरी है।
Enter the candidate's name and select the role before recording. This labels all saved files correctly.
2
Tap ⏺ Record — allow microphone
⏺ Record दबाएं — माइक की अनुमति दें।
iPhone users: open this in Safari and allow mic. The screen should stay on during recording. Android: use Chrome.
3
Audio auto-saves to Drive every 2 min
ऑडियो हर 2 मिनट में अपने आप Drive में सेव होता है।
Files are saved to a folder called "Enternetz Interview Lens" in your Google Drive. Nothing is lost if the interview runs long.
4
After interview: Analyse → Decide → Save
इंटरव्यू के बाद: Analyse करें → Decision लें → Save करें।
Run analysis, mark Shortlist / Hold / Reject, then Save to Drive. Full report + transcript go to Google Drive automatically.
5
Compare tab — see all candidates
Compare टैब में सभी उम्मीदवारों की तुलना करें।
The Compare tab loads all saved candidates from the shared Google Sheet — every panel member can see the same data.

PEB Role Scoring Guide
Detailer / Draftsman — CAD proficiency, drawing reading speed, attention to detail, PEB member knowledge (rafters, purlins, girts, bracing)
Senior Detailer — All above + connection details, anchor bolt plans, erection drawings, ability to work independently
Checker — Drawing accuracy, error identification, code awareness, cross-checking skills, communication with detailers
Senior Checker — All above + quality process ownership, mentoring ability, client drawing review experience
Team Leader — Leadership, deadline management, workload distribution, client coordination, escalation handling
